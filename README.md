# 2026 新高考英语冲刺工作台

这是一个面向全国通用新高考英语复习的静态网站原型。打开 `index.html` 即可使用，不需要安装依赖或启动服务。

## 已包含

- 2026 高考倒计时与目标分设置
- 今日冲刺任务清单
- 30/60/90 天冲刺计划切换
- 阅读、七选五、完形、语法填空、应用文、读后续写、词汇、听力专项入口
- 专项卡片包含常见失分点和训练动作
- 7 日冲刺日历，支持每日计划完成状态
- 每个专项 5 道微练题，支持上一题/下一题
- 专项限时练习、选项反馈和解析
- 答题统计、总体正确率和下一步建议
- 做错自动进入收藏回炉，也可手动收藏题目
- 微型模拟卷：答题卡、计时、写作草稿、交卷报告
- 模拟卷历史记录：保存最近 6 次成绩、正确率、作文词数和薄弱项
- 冲刺内容库：考前清单、写作素材、语法速查、考试节奏
- 写作提分实验室
- 写作任务库：应用文、通知、读后续写场景
- 写作自评清单：自动估算档位和 100 分制完成度
- 写作草稿自动保存和词数统计
- 草稿归档与载入，保存最近 5 次写作记录
- 高频主题词块
- 词块搜索与点击写入草稿
- 错题回炉卡片、自定义错题和收藏回炉队列
- 深浅色切换
- 本地学习进度保存
- 桌面端与移动端适配

## 本轮新增

- 提分处方包：按听力、阅读、语法、写作拆成 20-30 分钟可执行训练包
- 每个处方包包含目标、步骤清单、完成度、训练依据和今日产出
- 处方包可跳转对应专项练习，并在本地保存每一步完成状态
- 目标分拆解：根据目标分动态生成听力、阅读、语言运用、写作的分数预算
- 保分动作清单：把失分预算、证据复盘、时间上限和每日产出做成本地打卡
- 错因标签体系：收藏回炉题自动带错因，支持手动切换标签
- 错因雷达：学习报告聚合高频错因，并提供对应专项回练入口
- 智能周计划：根据目标分、剩余天数、专项统计和错因标签生成 7 天训练安排
- 周计划支持每日时长切换、完成打卡、专项跳转和生成依据说明
- 多套模拟卷：支持基础稳分卷、阅读强化卷、写作诊断卷切换
- 模拟错题自动入库：交卷后未答或答错的选择题会进入收藏回炉并带错因标签
- 模拟趋势分析：展示最近模拟正确率走势、分数变化和反复薄弱项
- 卷后处方跳转：从趋势面板直达最近薄弱专项或对应提分处方
- 模拟卷重做：可清空当前卷答案和作文草稿，同时保留历史趋势记录
- 内容扩容层：新增 `data/expansion.js`，把练习题、词块和资料卡从主逻辑中继续外置
- 题库补充：阅读、七选五、完形、语法、应用文、续写、词汇、听力各新增 2 道冲刺微练
- 素材补充：新增高频词块与复盘资料卡，覆盖模拟卷复盘、续写承接、动词空判断和周末节奏
- 证据化讲评：练习反馈支持定位句、干扰项、解题动作和错因标签，错题回炉同步保存这些讲评线索
- 核心题型加密：阅读、七选五、完形、语法各追加 2 道更接近高考讲评口径的分层题
